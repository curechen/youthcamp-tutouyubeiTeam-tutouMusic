# 秃头音乐

秃头预备选手专属音乐APP

## 安装依赖

> 将仓库下载到本地后运行该命令安装依赖

```bash
npm install
```

### 本地运行项目
```bash
npm run dev
```

### 打包构建
```bash
npm run build
```

### 项目基础配置

#### 推荐插件

在 `.vscode/extensions.json`中列出了推荐安装的插件，打开 VSCode 的时候会提示是否安装，最好全部安装一下。

#### workspace 设置

在 `.vscode/settings.json` 中配置了项目的共享配置，这些配置在所有的协作者本地都是一样的，保证大家的环境一致性。
